# Tetris
Tetris The Game

#### Windows:  
You just need Zig compiler  


#### Linux:
You need Zig compiler  
You need to install following library:  
`libsdl2-dev`  
`libsdl2-ttf-dev`  
`libsdl2-mixer-dev`  


```
git clone https://github.com/slier81/Tetris  
cd Tetris  
zig build play  
```
